About the System

The Plant Disease Detection System is an intelligent web-based application that predicts plant disease risk using environmental and soil-related parameters. It uses an Artificial Neural Network (ANN) for prediction.

🔍 Input Features

The model uses four environmental features:

  • Temperature (°C) – Air temperature affecting plant growth
  • Humidity (%) – Moisture level in the air
  • Rainfall (mm) – Amount of rainfall received
  • Soil pH – Acidity/alkalinity of the soil

🧠 AI Agents

The system uses three AI agents to process and interpret the ANN output:

  • 1. Confidence Agent: Assesses how reliable the prediction is (Low, Medium, High confidence).
  • 2. Disease Status Agent: Converts probability into categorical status: Healthy or Diseased based on a threshold.
  • 3. Risk Decision Agent: Converts probability into risk levels (Low, Moderate, High) and provides actionable recommendations for the user.

Together, these agents form an intelligent decision-making framework that guides users in assessing plant health risk.

🤖 How the Prediction Works

  1. User inputs environmental parameters (temperature, humidity, rainfall, soil pH).
  2. ANN model outputs a probability score indicating disease risk.
  3. AI agents interpret the probability:
    • Confidence agent shows prediction reliability.
    • Disease status agent determines Healthy/Diseased.
    • Risk decision agent provides risk level and recommendations.
  4. Results are displayed and saved in the database for historical tracking.

📊 Data Storage

All predictions are stored in a local database. Users can review historical results to track plant health trends over time.

🎯 System Objectives

  • Early detection of plant disease risk
  • User-friendly interface
  • Accurate predictions with minimal features
  • Decision support through intelligent recommendations
  • Track historical plant health trends
