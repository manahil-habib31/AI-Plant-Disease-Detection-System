# 🌱 AI-Based Plant Disease Detection System

An intelligent web-based application that predicts plant disease risk using environmental and soil parameters. The system uses an Artificial Neural Network (ANN) for prediction and AI decision agents to interpret the results, providing users with meaningful recommendations for plant health management.

---

## 📌 Project Overview

Plant diseases significantly affect agricultural productivity worldwide. Early prediction helps farmers take preventive measures before diseases spread.

This project uses Machine Learning and Artificial Intelligence to estimate disease risk based on environmental conditions such as:

- Temperature
- Humidity
- Rainfall
- Soil pH

Instead of simply predicting a probability, the system includes multiple AI agents that analyze the prediction and generate user-friendly recommendations.

# 🧠 Machine Learning Model

The prediction model is an Artificial Neural Network (ANN) developed using TensorFlow/Keras.

### Input Features

- Temperature (°C)
- Humidity (%)
- Rainfall (mm)
- Soil pH

### Output

Disease Probability

The probability generated by the ANN is interpreted by intelligent agents to provide:

- Disease Status
- Confidence Level
- Risk Level
- Recommendations

---

# 🤖 AI Agents

The application includes three intelligent AI agents.

## 1. Disease Status Agent

Converts ANN probability into:

- Healthy
- Diseased

---

## 2. Confidence Agent

Evaluates prediction reliability as:

- High
- Medium
- Low

---

## 3. Risk Decision Agent

Determines:

- Low Risk
- Moderate Risk
- High Risk

It also generates recommendations such as:

- Monitor plant condition
- Improve drainage
- Avoid overwatering
- Apply preventive spray

---

# 📊 Dataset

The ANN model was trained using a publicly available agricultural dataset obtained from Kaggle.

### Dataset Information

- Source: Kaggle
- Records: 10,000
- Features:

  - Temperature
  - Humidity
  - Rainfall
  - Soil pH

- Target:

  - disease_present (0 = Healthy, 1 = Diseased)

The dataset was preprocessed before training using feature scaling with StandardScaler.

# 📈 Prediction Workflow

1. User enters environmental parameters.
2. Data is scaled using StandardScaler.
3. ANN predicts disease probability.
4. Disease Status Agent classifies the result.
5. Confidence Agent evaluates prediction confidence.
6. Risk Decision Agent determines risk level.
7. Recommendations are generated.
8. Prediction is stored in SQLite database.
9. User can view prediction history.
